HTML <i> Tag


  Example :

  Mark up text that is set off from the normal prose in a document:

     <p>This is normal text - <i> and this is italic text</i>.</p> 

  Result: This is normal text - and this is italic text.





  Definition and Usage

  The <i> element makes the surrounded text display as italic.

  The <i> tag is often used to indicate a technical term, a phrase from another language, a thought, etc.
