HTML <ul> Tag


  Example :

  An unordered HTML list:

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

  Output:

  • Coffee                  
  • Tea
  • Milk




  Definition and Usage

  The <ul> tag tag defines an unordered (bulleted) list.

  Use the <ul> tag together with the <li> tag to create unordered lists.
