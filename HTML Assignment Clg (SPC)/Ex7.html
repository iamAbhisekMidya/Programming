HTML <h1> to <h6> Tag


  Example :

  The six different HTML headings:

     <h1>This is normal heading 1</h1> 
     <h2>This is normal heading 2</h2> 
     <h3>This is normal heading 3</h3> 
     <h4>This is normal heading 4</h4> 
     <h5>This is normal heading 5</h5> 
     <h6>This is normal heading 6</h6> 

  Output:


This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6





  Definition and Usage

  <h1> to <h6> tags are used to define HTML headings.

  <h1> defines the most important heading. <h6> defines the least important heading.
